# **App Name**: Al-Dirazi Elegance

## Core Features:

- Unified SPA Navigation: A seamless single-page layout featuring smooth-scroll transitions between five key sections using an anchored navigation bar.
- Digital Biography: A clean, typography-focused history of the reciter’s career and legacy, styled for premium readability.
- Dynamic Recitation Repository: A responsive audio library that fetches recitations from a Firestore database, featuring dedicated audio player cards with individual metadata.
- Mobile-Optimized Media Wall: A fully responsive video gallery embedding YouTube content into an adaptive grid that scales perfectly across devices.
- Generative Verses Assistant Tool: A search feature powered by AI that functions as a tool to help users find specific recitation segments based on spiritual themes or keywords.
- Integrated Contact Hub: A minimalist inquiry form combined with linked social media icons for professional outreach and listener engagement.

## Style Guidelines:

- The interface follows a premium dark theme. The primary gold color is #c5a059, creating a luxurious and spiritual contrast against a deep obsidian background (#0f0e0c) that shares the same warm hue. An analogous deep terracotta (#a3341a) serves as the accent color for active states and critical interactions.
- Note: currently only Google Fonts are supported. 'Tajawal' (Extra Light/Light) is used for all textual content, functioning as a elegant sans-serif that ensures a minimalist and serene aesthetic across both headings and body text.
- The application uses a centered-column layout with high vertical whitespace to create a sense of calmness, featuring fluid grid containers for audio and video media.
- Implementation of subtle fade-in effects on section arrival triggered by Intersection Observer for an ultra-sleek, immersive user experience.
- A set of thin-line, gold-tinted minimalist icons for playback controls and social media links, maintaining a light visual weight.
